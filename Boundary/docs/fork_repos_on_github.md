# Fork Repositories on GitHub

Before you can push to your forks, you need to create them on GitHub.

## Steps to Fork

### 1. Fork zashi-android

1. Go to: https://github.com/Electric-Coin-Company/zashi-android
2. Click the **"Fork"** button (top right)
3. Choose your account (`cczerp`) as the destination
4. Wait for the fork to complete

### 2. Fork zashi-ios

1. Go to: https://github.com/Electric-Coin-Company/zashi-ios
2. Click the **"Fork"** button (top right)
3. Choose your account (`cczerp`) as the destination
4. Wait for the fork to complete

## Verify Forks Exist

After forking, verify your forks exist:
- https://github.com/cczerp/zashi-android
- https://github.com/cczerp/zashi-ios

## Then Push Your Branches

Once the forks exist, you can push:

```powershell
cd android
git push -u origin boundary-android
```

```powershell
cd ios
git push -u origin boundary-ios
```
